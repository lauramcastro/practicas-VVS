Function Details

abrir/2

abrir(NClientes::integer(), Especialistas::[pid()]) -> ok

Starts the component. Receives the maximum number of allowed clients in the queue, which must be greater than zero. Receives the list of (created, started) specialists (references), which may not be empty.

cerrar/0

cerrar() -> ok

Shuts down the component. Since this component does not start the specialists, NOR does it take care of shutting them down.

entrar/1

entrar(Cliente::pid()) -> boolean()

Client admission. Receives a client (reference), which will be either accepted in (boolean value true is returned) if there is room, or dismissed (boolean value false is returned).
